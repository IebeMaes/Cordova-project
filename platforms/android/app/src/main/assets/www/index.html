Over de app

Dit is een utilities app waarin een plugins verwerkt zitten. Er is namelijk gebruik gemaakt van een toast-plugin en een in-app-browser-plugin. De in app browser wordt gebruikt bij het overzicht van het nieuws om het volledige artikel te tonen en de toast plugin wordt gebruikt tijdens het verzenden van het formulier. Verder zit er ook een api in verwerkt die de nodige informatie van het nieuws ophaalt. Dit is een project voor het vak Cordova van de IT Factory - Thomas More.
